build(backbones: Dict[str, sesemi.models.backbones.base.Backbone], heads: Dict[str, sesemi.models.heads.base.Head], **kwargs)

Builds the loss head.

Parameters

  • backbones – A dictionary of shared backbones. During the build, new backbones can be added. As this is actually an nn.ModuleDict which tracks the parameters, these new backbones should not be saved to the loss head object to avoid double-tracking.

  • heads – A dictionary of shared heads similar to the backbones
